01_基础_测试.md-检测题

题目1

vuex的作用是什么？

答案

对数据进行集中管理。

题目2

补全如下使用vuex时的伪代码：

import Vuex from 'vuex'
var store = new ________
export default store

把new后面缺少的代码补起来。

答案

参考如下：

var store = new Vuex.Store({
	state:{},
    getters:{},
    mutations:{},
    actions:{}
})
export default store
题目3

如何理解vuex中的state？

答案

它是一个对象，用来保存所有的公共数据。

题目4

在组件内部获取vuex中的数据有哪些方法？

答案

方法一：直接通过this.$store.state.数据来获取。

方法二：借用mapState，把数据映射成组件内的计算属性后，再来使用。

方法二：在vuex中创建getter，然后在组件内部通过getter来获取。
