题目4

URL是什么？有使用用处？基本构成是什么？

答案

Uniform Resource Locator，统一资源定位符。

用于标识互联网上每个资源(css,html,js,png, avi......)的唯一存放位置。

它一般由三个部分组成：通信协议,服务器名称,服务器上的具体位置。

题目10

请写出$.ajax()的调用格式，并说明每个参数的含义。

答案 
$.ajax(
  {
	 url,   // 请求接口地址
     type,  // 请求的类型
     data,  // 携带的数据
     success: function (result) {
     //回调函数。当请求成功之后会自动调用这个函数，它的第一个参数是服务器端返回的数据。
     	console.log(result)
  	 }
  }
)

题目15

观看如下代码：

$.ajax({
   url: 'https://api.com/getinfo',   // 请求接口地址
   type: 'get',  // 请求的类型
   success: function (res) {
	 if(res.status === 200) {
         console.log(res.data)
     }       
   }
})

请回答：上面的代码中的200是什么？ 可以改成其它的数值吗？

答案

res表示一个对象，它是后端接口返回的数据。其中有一个属性名是status，它的值表示这个接口的工作状态。一般前后端同学会约定200表示本次接口工作正常，注意这个值是由后端同学定的，也可能是其它的数值。
