题目1

实现ajax功能的那个对象是？

答案

题目2

使用原生ajax的基本步骤是什么，请写出伪代码。

答案 
// 第1步： 新建一个XMLHttpRequest对象(xhr)
var xhr = new XMLHttpRequest();

// 第2步： 设置请求。 
// 格式：xhr.open(请求的方式, 请求的url地址);
xhr.open('请求类型', '接口地址');

// 第3步：设置回调函数

xhr.onload = function() {
    console.info('从服务器返回的数据',xhr.responseText);
};
// 第4步： 发送请求
xhr.send();

题目4

在用原生ajax时，使用post类型的接口与get类型的接口有何区别？

答案

get类型的接口参数以查询字符串格式附在url中。

post类型的接口写在send()中，要明确设置请求头。

题目11

写函数，把对象转成查询字符串

function f(obj){
    // 你的代码
	// return {}  
}
// 测试用例：
console.info(f({a:1,b:2,c:jame})); 
// 得到 'a=1&b=2&c=jame'
答案 
 function f(obj){
     var arr = [];
     for (var key in obj) {
         // 找出对象的全部的属性
         arr.push(key + '=' + obj[key]);
     }
     return arr.join("&") 
}

题目16

http状态码401，403的区别。

答案

401：您的Web服务器认为，客户端发送的 HTTP 数据流是正确的，但进入网址 (URL) 资源 ， 需要用户身份验证 ， 而相关信息 可能：尚未被提供 或 已提供但没有通过授权测试。

403：您的Web服务器理解了本次请求但是拒绝执行该任务。可能是您没有这个操作权限。
